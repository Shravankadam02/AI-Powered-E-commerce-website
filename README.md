# 🛒 AI-Powered E-Commerce Platform

A modern full-stack e-commerce platform enhanced with AI-driven features to deliver a smarter and more personalized shopping experience.

This project focuses on combining traditional e-commerce functionality with intelligent systems such as smart search, recommendations, and user-centric interactions.

---

## ✨ Features

* 🔐 Authentication (including Google OAuth)
* 🛍️ Product browsing and collections
* 🛒 Cart management system
* 👤 User profile & order tracking
* 🔍 Smart search interface
* 📱 Fully responsive UI (desktop + mobile navigation)
* ⚡ Fast and optimized frontend with Vite

---

## 🧠 AI Capabilities

* Intelligent product discovery
* Personalized user experience
* Scalable architecture for recommendation systems

AI is increasingly transforming e-commerce by improving personalization, product discovery, and customer interaction across the shopping journey

---

## 🛠️ Tech Stack

### Frontend

* React (Vite)
* Tailwind CSS
* React Router
* Context API
* React Icons

### Backend

* Node.js
* Express.js
* MongoDB (Mongoose)

## ⚙️ Installation & Setup

```bash
# Clone repository
git clone https://github.com/Shravankadam02/AI-Powered-E-commerce-website.git

# Frontend setup
cd Frontend
npm install
npm run dev

# Backend setup
cd ../Backend
npm install
npm run dev
```
